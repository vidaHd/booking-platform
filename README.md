# Booking Platform

A multi-service online reservation platform that allows service providers to manage offerings and users to book available time slots seamlessly.

## Features
- Users can view available services and book time slots.
- Service providers can manage services, approve reservations, and receive real-time notifications.
- Real-time email notifications for booking confirmations.
- Frontend built with React.js and Tailwind CSS.
- Backend built with Node.js, Express.js, and MongoDB.
- Dockerized for easy deployment.

- Frontend: React.js, TypeScript, Tailwind CSS
- Backend: Node.js, Express.js, MongoDB
- Real-time: Socket.IO
- DevOps: Docker, Git
